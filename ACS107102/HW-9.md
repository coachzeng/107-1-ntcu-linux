##### 安裝軟體

### 閱讀 `https://www.cyberciti.biz/faq/installing-rhel-epel-repo-on-centos-redhat-7-x/` 後，依照其作法，啟用epel reposisitory，並安裝`htop`
1.  `yum install epel-release`
![image](https://github.com/acs107102/107-1-ntcu-linux/blob/acs107102-patch-1/linux/HW9-1.JPG)
2.  `sudo yum repolist`
![image](https://github.com/acs107102/107-1-ntcu-linux/blob/acs107102-patch-1/linux/HW9-2.JPG)
3.  `sudo yum search htop`
![image](https://github.com/acs107102/107-1-ntcu-linux/blob/acs107102-patch-1/linux/HW9-3.JPG)
4.  `sudo yum info htop`
![image](https://github.com/acs107102/107-1-ntcu-linux/blob/acs107102-patch-1/linux/HW9-4.JPG)
5. `sudo yum install htop`
![image](https://github.com/acs107102/107-1-ntcu-linux/blob/acs107102-patch-1/linux/HW9-5.JPG)
